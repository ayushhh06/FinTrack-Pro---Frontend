# FinTrack Pro - Frontend

React + Tailwind CSS UI for expense tracker application.

## Tech Stack
- React 18+
- Vite
- Tailwind CSS
- Recharts
- Lucide React Icons

## Setup
1. Install Node.js 16+
2. Run: `npm install`
3. Run: `npm run dev`

## Features
- User authentication
- Add/delete expenses
- Expense visualization (charts)
- Category tracking
- Real-time metrics
